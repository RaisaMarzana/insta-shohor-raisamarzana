How does Javascipt works?

Javascript is embedded in html using html element script tag.Javascript code can be written directly inside the script tag. Or it can be written in a dedicated file that link to html documunet. It is a typical programming language. It has native asynchronous mechanism that let it react to events.

What does Javascript event loops do?

The event loop is the JavaScript asynchronous programming. JS executes all operations on a single thread, but using a few data structures, it gives the illusion of multi-threading.

Difference between local storage and session storage.

Local storage: Any kind of data can be stored in local storage. There is no expiration time for data in local storage even the browser is closed. Session storage: Any kind of data can be stored only for that particular session. Session storage has an expiration time. Data will be lost when browser is closed.
